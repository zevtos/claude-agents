# Формулы (LaTeX)

Метод `r.formula(latex, where=None) → int` вставляет формулу как **нативное Word-уравнение** (OMML) — открывается, редактируется и масштабируется в Word без сторонних инструментов. Под капотом: LaTeX → MathML (через `latex2mathml`) → OMML (собственный walker, чистый Python).

Метод возвращает `int` — номер формулы — для ссылок из текста:
```python
f1 = r.formula(r"E = mc^2")
r.text(f"По формуле ({f1}) видна связь массы и энергии.")
```

Формула центрируется, номер «(N)» прижат к правому краю на той же строке (по ГОСТ 7.32). Опциональный `where=...` рендерит блок «где: ...» под формулой обычным абзацем.

## Поддерживается

Всё, что парсит `latex2mathml`:

| Конструкция | Пример |
|---|---|
| Степени, индексы | `x^2`, `a_{ij}`, `x_i^j` |
| Дроби | `\frac{a+b}{c}` |
| Корни | `\sqrt{x}`, `\sqrt[3]{x^2 + y^2}` |
| Греческие буквы | `\alpha`, `\beta`, `\Gamma`, `\theta`, ... |
| N-арные с пределами | `\sum_{i=1}^{n}`, `\int_0^\infty`, `\prod_{k=1}^{m}`, `\oint` |
| Акценты | `\bar{x}`, `\hat{y}`, `\vec{F}`, `\tilde{q}`, `\dot{p}`, `\ddot{q}`, `\acute`, `\grave`, `\check`, `\breve` |
| Матрицы | `\begin{pmatrix}...\end{pmatrix}`, `bmatrix`, `vmatrix`, `Vmatrix`, `matrix` |
| Текст внутри | `\text{...}` |
| Скобки | `()`, `[]`, `\{\}`, `\left|\cdot\right|` |
| Операторы | `\cdot`, `\times`, `\pm`, `\leq`, `\geq`, `\neq`, `\approx`, `\equiv` |
| Пробелы | `\,`, `\;`, `\quad`, `\qquad` |

## Не поддерживается

- **`\begin{align}`, `\begin{equation}`, `\begin{eqnarray}`** — это макросы окружений, `latex2mathml` их не парсит. Каждая формула вызывается **отдельно** через `r.formula(...)`. Если нужна цепочка равенств в одной формуле, пиши их линейно: `r.formula(r"a = b = c")`.
- **Кастомные макросы** (`\newcommand`, `\def`).
- **`\usepackage`-зависимое.**
- **Многострочные формулы.** Каждая `r.formula()` это одна строка-формула. Для последовательности равенств — несколько вызовов с одним номером каждый.

Если попадётся неподдерживаемая конструкция — `latex2mathml` либо бросит ошибку, либо отрендерит как текст. Проверяй результат в Word.

## Примеры

### Простая степень
```python
r.formula(r"E = mc^2")
```

### Дробь с пояснением переменных
```python
r.formula(
    r"\frac{\partial u}{\partial t} = \alpha \nabla^2 u",
    where="u температура, α коэффициент диффузии, t время.",
)
```

### Сумма с пределами
```python
r.formula(r"\sum_{i=1}^{n} i = \frac{n(n+1)}{2}")
```

### Интеграл
```python
r.formula(r"\int_0^\infty e^{-x^2}\, dx = \frac{\sqrt{\pi}}{2}")
```

### Векторы и акценты
```python
r.formula(r"\vec{F} = m\vec{a}")
r.formula(r"\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i")
```

### Матрица
```python
r.formula(r"A = \begin{pmatrix} a_{11} & a_{12} \\ a_{21} & a_{22} \end{pmatrix}")
```

### Цепочка ссылок на формулы
```python
f_main = r.formula(r"y = ax + b")
f_inv  = r.formula(r"x = (y - b)/a")
r.text(f"Уравнение ({f_main}) описывает прямую, обратное к нему ({f_inv}).")
```

## Санитайзер и LaTeX

Строка `latex` идёт в `latex2mathml` **сырой**, без `_sanitize_prose`. Это специально: `\text{a — b}` или диапазон `[1—5]` сломали бы LaTeX-синтаксис. Поэтому **внутри** LaTeX-строки можно (и иногда нужно) использовать любые символы, включая тире.

А вот аргумент `where=...` это обычная русская проза, к ней применимо ⛔ Правило №1 из SKILL.md: **никаких длинных тире**. Санитайзер заменит ` — ` на `, `, но лучше писать сразу без тире:

```python
# Плохо (санитайзер сработает, но это надежда на страховку):
r.formula(r"...", where="a, b — слагаемые, c — делитель")

# Хорошо (без тире, читается естественнее):
r.formula(r"...", where="a, b слагаемые, c делитель")
```

## Шрифт и стиль внутри формул

OMML-walker автоматически расставляет:
- Одиночные латинские/греческие буквы в `<mi>` → курсив (математический дефолт).
- Многобуквенные идентификаторы (`sin`, `log`, `lim`) → прямой шрифт.
- Числа `<mn>` и операторы `<mo>` → прямой шрифт.

Если нужно явно прямой шрифт для одной буквы — используй `\mathrm{x}`. Если нужен текстовый блок внутри формулы — `\text{констант}`.

## Что делать, если в Word формула выглядит странно

1. Открой файл в Word (не PDF preview, не LibreOffice — для формул нужен именно Word или совместимая программа).
2. Если видна сама LaTeX-строка как текст — значит, `latex2mathml` упал. Проверь синтаксис LaTeX, запусти `latex2mathml.converter.convert(...)` отдельно.
3. Если видны квадраты-placeholder'ы между знаком ∑/∫ и формулой — это пустое тело `<m:e>`. Проверь, что после оператора в формуле есть подынтегральное выражение.
4. Если стрелка `\vec{F}` пересекает букву — это spacing/combining проблема, должно быть пофикшено в v0.5.0+. Проверь версию скилла.
